# Convert a Word document table into an email message

For teams who create tables in Word documents and need to send the table content as an email without losing the formatting.

## Steps

1. Open the Word document that contains the table you want to send.
2. Click on the table to select it (you should see selection handles around the edges).
3. Copy the entire table (Ctrl+C).
4. Open your email application (Outlook, Gmail, or similar).
5. Click Compose to start a new email message.
6. In the email body field (the large text area), click to place your cursor where you want the table to appear.
7. Paste the table (Ctrl+V). The table should appear with all its rows and columns.
8. If the table does not look right or is too wide for an email, you can type the content manually or simplify it: delete columns you do not need, or convert the table to plain text with commas or dashes separating the values.
9. Add a subject line for the email (e.g., "Q1 2024 Sales Report" or "Team Attendance Summary").
10. Type any message above or below the table to explain what the table contains.
11. Enter the recipient email address(es) in the To field.
12. Click Send.

## Inputs

| Name | What it is | Example |
|------|-----------|---------|
| Word document | File containing the table to send | monthly-report.docx |
| Table name or location | Which table in the document if there are multiple | "Sales by Region" table on page 2 |
| Email recipients | Who should receive the email | manager@company.com, team@company.com |
| Email subject | Message subject line | Q1 2024 Sales Summary |

## The workflow file goes here

Once the workflow is captured, the file will be saved as a workflow file that you can run again without retyping these steps.
