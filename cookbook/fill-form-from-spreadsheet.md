# Fill a form from a spreadsheet row

For data entry teams who need to take information from an Excel or Google Sheets file and enter it into web forms, one row at a time.

## Steps

1. Open your spreadsheet file (Excel or Google Sheets) that contains the data to be entered.
2. Open your web browser and navigate to the form you need to fill out.
3. Look at the first row of data in the spreadsheet (skip the header row with column names).
4. In the web form, find the first field and compare its label with your spreadsheet column headers.
5. Copy the cell from the spreadsheet (Ctrl+C) that matches that field.
6. Click on the web form field and paste the data (Ctrl+V).
7. Tab or click to move to the next field in the form.
8. Repeat steps 4-7 for each field in the form until the entire row of data has been entered into the form.
9. Look for a Submit or Save button on the form and click it.
10. Note the confirmation message (if any) to know the form was sent successfully.
11. Go back to the spreadsheet, move down to the next row, and repeat steps 4-10 for each row of data.

## Inputs

| Name | What it is | Example |
|------|-----------|---------|
| Spreadsheet file | Excel or Google Sheets with data to enter | customer-data.xlsx |
| Web form URL | Web address of the form to be filled | company.com/registration-form |
| Field mapping | How spreadsheet columns match form fields | Column A = Name field, Column B = Email field |
| Data range | Which rows of data to process | Rows 2 through 50 |

## The workflow file goes here

Once the workflow is captured, the file will be saved as a workflow file that you can run again without retyping these steps.
