# Organize files by type and move them into folders

For anyone with a messy folder full of mixed file types who needs to sort them into organized subfolders.

## Steps

1. Open File Explorer and go to the folder that needs organizing.
2. Look at the file types you see (e.g., .pdf, .xlsx, .docx, .jpg, .txt, etc.) and decide on folder names (e.g., "PDFs", "Spreadsheets", "Documents", "Images", "Other").
3. Create a new folder for each file type: right-click in the empty space, select New > Folder, and type the folder name.
4. Click on the first file and check its file type by looking at the extension after the period (e.g., Invoice.pdf is a PDF file).
5. Drag that file into the matching folder (e.g., drag the PDF into the "PDFs" folder).
6. Move to the next file and repeat steps 4-5 until all files have been sorted.
7. For files you are unsure about, move them to an "Other" folder or skip them for now and ask someone later what they are for.
8. Once all files are organized, close File Explorer and take a screenshot of the folder structure to confirm it looks correct.

## Inputs

| Name | What it is | Example |
|------|-----------|---------|
| Source folder | Folder with mixed files to organize | C:\Users\Name\Documents\Archive |
| File types | Categories of files to sort | PDF, Excel, Word, Images, Text |
| Folder names | Names for the new subfolders | PDFs, Spreadsheets, Documents, Images |
| Action on conflict | What to do if a file type folder already exists | Merge files into existing folder |

## The workflow file goes here

Once the workflow is captured, the file will be saved as a workflow file that you can run again without retyping these steps.
