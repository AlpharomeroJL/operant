# Cookbook: Real-World Workflows

Ten examples of tasks people do every day, explained step-by-step in plain English.

1. **Copy invoice rows from a web portal into a spreadsheet**: Move data from a vendor or customer portal into Excel without retyping. (Benchmark)

2. **Rename and file downloaded PDFs by date**: Organize downloaded PDFs into date-based folders with clear file names. (Benchmark)

3. **Reply to a canned customer email**: Send the same response to multiple similar support requests without typing it each time.

4. **Extract totals from a scanned receipt**: Pull dollar amounts and dates from receipt images into an expense spreadsheet.

5. **Fill a form from a spreadsheet row**: Take information from Excel or Google Sheets and enter it into web forms, one row at a time.

6. **Download daily reports and combine them into one file**: Collect daily reports from a website and merge them into a single spreadsheet for review.

7. **Convert a Word document table into an email message**: Copy a table from Word into an email while keeping its formatting.

8. **Organize files by type and move them into folders**: Sort a messy folder full of mixed files into organized subfolders by type.

9. **Extract and format text from images for a document**: Copy text out of images (whiteboard photos, screenshots, scanned pages) into a document. (Benchmark)

10. **Send personalized emails using a contact list**: Send similar emails to multiple people with each person's name and details filled in.
