# Extract and format text from images for a document

For researchers, students, and professionals who need to copy text out of images (such as photos of whiteboards, screenshots, or scanned pages) and put it into a document.

## Steps

1. Have ready the image files that contain text you want to extract (they may be on your desktop, in a folder, or emailed to you).
2. Open the first image to look at it and see what text needs to be copied.
3. Read the text in the image carefully and look for the context: are these bullet points, sentences, a list, a table, or something else.
4. Open a blank document (Word, Google Docs, Notepad, or similar) where you want the extracted text to go.
5. Type or paste the text from the image into the document, organizing it in the same structure as it appeared in the image (e.g., if it was a bulleted list, make it a bulleted list in the document).
6. For sections of text that are hard to read in the image, type your best guess and add a note in brackets to mark uncertain words (e.g., "[unclear]" or "[possibly 'example']").
7. Once the text from the first image is entered, move to the next image and repeat steps 2-6.
8. After all images have been processed, read through the document once to check for typos or words you need to fix.
9. Save the document with a clear name that explains where the text came from (e.g., "Whiteboard-notes-2024-01-15.docx").

*Benchmark: Yes*

## Inputs

| Name | What it is | Example |
|------|-----------|---------|
| Image files | Source images containing text to extract | Photos, screenshots, scanned pages |
| Document format | Where to paste the extracted text | Word document, Google Doc, or text file |
| Text type | How the text is organized | Bullet points, sentences, table, notes |
| Output location | Where to save the final document | C:\Users\Name\Documents\Extracted-Text |
