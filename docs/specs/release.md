# Spec: release

NSIS single-source bundle; per-user install (no admin) by default with a machine-install option; uninstaller removes binaries and PATH entries but PROMPTS before touching user data (workflows, recordings). Updater: manifest at a stable URL, Ed25519-signed, staged download, verify, swap on next start; endpoint present in release profile asserted by CI (a release build is scanned for the endpoint constant); private key generated in Phase 0 to the vault path in release/KEYS.md and never committed (CI greps history). SBOM via cargo-auditable plus npm ls. Reproducible build doc: pinned toolchains, vendored lockfiles, one-command rebuild. gh release with installer, SHA256SUMS, SBOM, notes from DECISIONS lines; post-upload signature re-verification by re-download.
