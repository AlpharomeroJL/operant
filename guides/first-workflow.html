Your First Workflow

Build and save your first automated workflow in 15 minutes.

What You'll Learn

By the end of this guide, you will have created, saved, and run your first workflow. No typing, no coding required. This takes about 15 minutes.

Before You Start

Pick a task you do by hand repeatedly. Some examples:

  • Copy data from a website into a spreadsheet.
  • Fill in a form with the same information each time.
  • Rename files based on their content.
  • Move files from one folder to another.

Step 1: Open Operant and Start Teaching

Click the Operant icon in your system tray. Select "Record New Workflow". Operant will ask you to describe what you want to teach (you can type or use your microphone).

Step 2: Do It Once, and Operant Watches

Operant watches as you perform your task on your computer. Open the apps you need, click buttons, type text, move files, whatever you normally do. Operant is recording everything you do. You can talk out loud if you want ("Now I copy the invoice number from the website").

Step 3: Save It as a Workflow

When you finish, click "Save as Workflow". Operant will show you what it learned: a numbered list of steps in plain English. You can edit any step, fix mistakes, or add parameters like "which file to copy" or "where to paste it".

Step 4: Run It Again

Click "Run" and watch Operant do it all over again, this time without you touching the mouse or keyboard. It runs fast because Operant now knows exactly what to do.

Step 5: Schedule It (Optional)

If you want it to run by itself, click "Schedule" and choose when: every Monday at 9 AM, every time a file arrives in a folder, when you receive an email, or whenever you click a button. Read the Scheduling guide for more options.

Troubleshooting

If the workflow does not work perfectly the second time (maybe a button moved or a form looked different), Operant will show you what changed and ask if you want to fix it. You can approve the fix in one click and move on.

What Happens Next?

Your workflow is now saved and ready to use whenever you need it. You can run it again, edit it, share it, or schedule it. In the "Scheduling" guide, learn how to set up your workflow to run automatically.
