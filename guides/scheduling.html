Scheduling Workflows

Automate your workflows with cron, file watches, and event triggers.

Trigger Types

Operant supports multiple ways to trigger compiled workflows:

  • Cron: Schedule recurring runs at specific times or intervals
  • File Watch: Trigger when a file in a directory changes
  • Window Appears: Trigger when a specific application window opens
  • Email Arrives: Trigger when a new email lands in your inbox
  • Manual: Run on demand from the tray or CLI

Setting Up Cron Triggers

Open the workflow and select "Schedule". Enter a cron expression like 0 9 * * MON to run every Monday at 9 AM.

Safety Limits

Unattended triggers may only run compiled workflows. Exploratory runs always require manual approval.

Monitoring Scheduled Runs

View execution history, logs, and any failures in the Operant dashboard.
