---
name: operant-ux
description: Zero-code experience rules for any packet producing user-facing UI or copy.
---
The user is a smart adult who has never coded and never will. Default mode shows no
jargon: lint every string against contracts/microcopy_glossary.json. Every error states
what happened, why, and one suggested action in one sentence each; add a one-click fix
when automatable. Workflows render as numbered plain-English steps with form-field
parameters; code lives behind the Advanced toggle only. Grants render as plain sentences
("This workflow can read files in Downloads and control Chrome"). Drift prompts are
offers, not diffs ("The Save button moved. Update the workflow?" with preview). Every
new surface must be reachable and completable by the first-timer path E2E in default
mode: no terminal, no code, no config file.
